What does it do ?


The eGenVar system provides a easy and systematic way to manage and advertise data.  A set of tags and relationships between the donors, samples and files are used when locating information


  1. Organise data
    • Record the relationships between different files using lineage instead of relying on folders
    • Record the content description using tags selected from Ontologies (the controlled vocabulary terms)
    • Organise all related data , regardless of their location using reports
    • View all private, public, unpublished and data residing in collaborators computers as a report batch
    • Using standardised metadata make it possible anyone to find what they are looking for not only the person created it
  2. Link
    • Linik the different files using lineage
    • Link to files to samples
    • Use tags to get a system which is more than the collection of parts
  3. Advertise
    • Tell the other researchers what you have
    • Makes it possible for you to find what you are looking for.
  4. Locate
    • Find controls
    • Avoid repeating the same thing or errors
    • Find experts t o collaborate
    • Discover hidden data, kept in that way due to privacy reasons
