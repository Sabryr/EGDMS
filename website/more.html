Private search - Browse

  • If you do not know the exact tag to be searched or the search is more general.
  • Select the "All tags" in search option and perform the search as usual.
  • This will search all tag sources (not only the ones that are used) and gives you matching results
  • Then you could investigate the interesting once for usage. 
  • For example, you could search for the term "Cancer" and investigate the usage of different cancer types as tags.
