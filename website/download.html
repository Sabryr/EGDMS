Download

 Server

There are two versions of the server. one for Unix /Linux and other for Windows. Both are same, except the way they are installed.

Click on the icon to download the Unix/Linux or the Windows version.


Installation instruction

 Clients

The clients are unique to each server instance. They should be downloaded from the server instance they are trying to connect. So if you are interested in using a client to connect to a existing eGenVar server, then you need to find the web address of that server. If you want to use it locally (e.g. on your PC) then install the server and the clients wil l be generated in the process.
